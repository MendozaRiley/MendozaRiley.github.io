Exercise 6


For this exercise, I converted my Processing code for Project 1 to p5.js. While I kept many of the same drawing tools, I added a bezier curve to the rmQuills custom function and a new custom function, rmAlien2, that employs shearX().

Custom Function's:

  • rmQuills: This is the function for the black and white lines that represent tree/grass like shapes of the alien sighting scene.
  • rmStars: This is the function for the random small white circles scattered around the sketches.
  • rmAlien1: This is the function for the alien figure that radiates light blue and purple circles from its core.
  • rmAlien2: This is the function for the alien figure made up of translucent quadrilaterals.
  • rmTopBot: This function contains the code for the vertical circles of the alien figure.
  • rmLeftRight: This function contains the code for the horizontal circles of the alien figure.
  • rmUFO1: This is the function for the first UFO shape that "comes in piece."
  • rmUFO2: This is the function for the second UFO shape with the white light emanating from the bottom.
  • rmUFO3: This is the function for the third UFO shape that emits the blue-ish green rays that will "destroy us all."
  • rmUTop: This function contains the code for the top halves of the UFOs.
  • rmUBot: This function contains the code for the bottom halves of the UFOs.

Project 3


For this project, I wanted to continue with the space theme from past projects. I decided to portray the passage of time through a concise, and fictional version of the life of a star. I used the hour(), minute(), second(), and map functions to control almost all of the aspects of this project, such as the shifting size and color of the center star and the commencement of the explosion. I used the push, pop, translate, angleMode, rotate, and scale to create most of the imagery for the explosion of the star. While I really like the end scene imagery of the explosion, I believe the project can be improved by adding more visual details to the earlier hours that seem more slow and less interesting.

Custom Function's:

  • rmStar: This is the function for the center star that explodes every day.
  • rmSupernova1: This function contains the code for the first part of the star explosion.
  • rmSupernova2: This function contains the code for the second part of the star explosion.
  • rmSupernova3: This function contains the code for the final part of the star explosion.
  • rmLight: This function contains the code for the object radiated in rmSupernova2.
  • rmRays1: This function contains the code for the object radiated in rmSupernova1.
  • rmRays2: This function contains the code for the object radiated in rmSupernova1.
  • rmStars: This is the function for the random small white circles scattered around the sketches.
  • rmPlanet1: This is the function for the planet with one outer ring.
  • rmPlanet2: This is the function for the planet with two small craters.
  • rmPlanet3: This is the function for the planet with two horizontal rings.
  • rmUFO: this is the function for the UFO that passes through the scene once.
  • rmUTop: This function contains the code for the top half of the UFOs.
  • rmUBot: This function contains the code for the bottom half of the UFOs.
