"Out of this World"

"Out of this World," Java, 2021

My initial approach to this project was exploratory, similarly to the previous sketches. I was testing out certain functions and playing around with pieces of code from the past exercises when I came up with the extraterrestrial theme. At first, I was planning on creating drawing tools that would allow the user to sketch an extraterrestrial sighting scene. However, as I began creating my own sketches, I strayed away from the more representational drawing plan and began using the drawing tools objectively. While the subject matter of my tool is space themed, the tool seems to work well with abstract art styles involving a cool color scheme and a variety of geometric and organic shapes. Regarding the success of the tool, its usage as a tool for both representational and abstract art styles was a pleasant surprise, yet it could have been improved if there was more movement after the shapes were drawn to really bring the sketch to life.

Custom Function's:

  • rmQuills: This is the function for the black lines that represent tree/grass like shapes of the alien sighting scene.
  • rmStars: This is the function for the random small white circles scattered around the sketches.
  • rmAlien1: This is the function for the alien figure that radiates light blue and purple circles from its core.
  • rmTopBot: This function contains the code for the vertical circles of the alien figure.
  • rmLeftRight: This function contains the code for the horizontal circles of the alien figure.
  • rmUFO1: This is the function for the first UFO shape that "comes in piece."
  • rmUFO2: This is the function for the second UFO shape with the white light emanating from the bottom.
  • rmUFO3: This is the function for the third UFO shape that emits the blue-ish green rays that will "destroy us all."
  • rmUTop: This function contains the code for the top halves of the UFOs.
  • rmUBot: This function contains the code for the bottom halves of the UFOs.

Project 2

Project 2, Java, 2021

I wanted to continue with the extraterrestrial theme for this project as with my previous sketches. However, I decided to take the scene out to space rather than viewing the night sky from Earth. The sketch features a small set of planets, stars, otherworldly rays of energy, and hidden UFOs. I incorporated the Springs algorithm for the movement of the planets as they follow the mouse while bouncing around. I employed millis() in order to create three sections. The first begins with white rays of energy, transforming into black rays in the second section, and returning to white rays in the third. I employed frameCount to control the UFO that appears every 100 frames in a random position. While I am happy with the aesthetic qualities of my project, I want to keep exploring more options with the code to create even more variation and complexity.

Custom Function's:

  • rmStars: This is the function for the random small white circles scattered around the sketches.
  • rmPlanet1: This is the function for the planet with the one ring at its center.
  • rmPlanet2: This is the function for the planet with the two small craters.
  • rmPlanet3: This is the function for the planet with the two rings.
