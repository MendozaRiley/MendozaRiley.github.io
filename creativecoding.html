Creative Coding

"Out of this World"

"Out of this World," Java, 2021

My initial approach to this project was exploratory. I was testing out certain functions and playing around with pieces of code from past sketches when I came up with the extraterrestrial theme for this drawing tool. At first, I was planning on creating a tool that would allow the user to sketch an extraterrestrial sighting scene. However, as I began creating my own sketches, I strayed away from the more representational drawing plan and began using the drawing tools objectively. While the subject matter of my tool is space themed, the tool seems to work well with abstract art styles involving a cool color scheme and a variety of geometric and organic shapes.

Custom Function's:

  • rmQuills: This is the function for the black lines that represent tree/grass like shapes of the alien sighting scene.
  • rmStars: This is the function for the random small white circles scattered around the sketches.
  • rmAlien1: This is the function for the alien figure that radiates light blue and purple circles from its core.
  • rmTopBot: This function contains the code for the vertical circles of the alien figure.
  • rmLeftRight: This function contains the code for the horizontal circles of the alien figure.
  • rmUFO1: This is the function for the first UFO shape that "comes in piece."
  • rmUFO2: This is the function for the second UFO shape with the white light emanating from the bottom.
  • rmUFO3: This is the function for the third UFO shape that emits the blue-ish green rays that will "destroy us all."
  • rmUTop: This function contains the code for the top halves of the UFOs.
  • rmUBot: This function contains the code for the bottom halves of the UFOs.

Sketch 3

Sketch 3, Java, 2021

For this sketch, I created my own Class for UFOs. I used similar imagery as Sketch 2 with my "Out of this World"custom functions, however, I added an abducted cow to the scene to further differentiate each of the UFOs and their intentions on Earth. The four methods of the UFO class include hover, abduct, attack, partytime, and invasion which includes an array. I use mousePressed and keyPressed to separate the imagery for each method so that their actions are distinct.

"Astro"

"Astro", Java, 2021

I wanted to continue with the extraterrestrial theme for this project as with my previous sketches. However, I decided to take the scene out to space rather than viewing the night sky from Earth. The sketch features a small set of planets, stars, otherworldly rays of energy, and hidden UFOs. I incorporated the Springs algorithm for the movement of the planets as they follow the mouse while bouncing around. I employed millis() in order to create three sections of the piece. The first begins with white rays of energy, transforming into black rays in the second section, and returning to white rays in the third. I used frameCount to control the UFO that appears every 100 frames in a new position.

Custom Function's:

  • rmStars: This is the function for the random small white circles scattered around the sketches.
  • rmPlanet1: This is the function for the planet with the one ring at its center.
  • rmPlanet2: This is the function for the planet with the two small craters.
  • rmPlanet3: This is the function for the planet with the two rings.
